# urvi-agent
A fully functional AI agent, developed as a personal assistant for URVI Montessori.


The ideas we have:
-> Personalized application and enquiries assistant: Helps with filling admission forms by breaking down each question, answer basic questions, and if they have doubts while they fill it out, we answer softly and appropriately; act like a guide to our virtual environment or be a picture-guide, where if a parent has questions about a specific environment, they can directly refer to us. The agent will also help with scheduling enquiries, push a form with response-based questions like, "Specific notes for the meeting"; "Any additional comments?", with a calendly interface for them to book the slot. 

-> Triggered Communication: Automated texts, emails if student is absent for the day, or if a parent requested for a query, timely updates with their application. Automated emails like admission success.. "Congralutations, welcome to URVI Montessori"; or our newsletters about Montessori information and fun facts; reminders or invitations to events we would conduct, etc etc. Timely updates about enquiries. 

-> 24/7 Request Panel: Any requests, questions or messages.. if came at an unlikely time, we always accept them and prompt them to wait for a certain amount of time until one of us reaches out to them personally. AI chatbot will handle questions that are generic or those for which it has the data to answer effectively.
